HTML Tags Reference

HTML Tags

Tag Name Description
<html> HTML Used to make the webpage appear appropriately.
<head> Head Contains the information about the document, such as webpage title, metatags, and css styles.
<title> Title Sets the title for the webpage to be seen in the browser tab.
<link> Link Allows you to link an external CSS stylesheet to the webpage.
<style> Style Allows you to add CSS styles to the webpage.
<body> Body Contains all of the content that will appear on the webpage.
<h1> - <h6> Heading 1 - Heading 6 Formats the text to be a heading, with <h1> being the largest sized text and <h6> being the smallest sized text.
